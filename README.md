# Knowledge-base-Q-A-iteration

## get question
基于一个文件夹批量生成QA，会依次检索文件夹内的信息，亲测txt和无格式文件有效
## get answer
基于xlsx文件生成问题，文件至少含问题、原文和来源表头
## improve answer
基于xlsx文件提示QA质量，文件至少含问题、原文和答案3个表头

## LLM
已经给出Openai,Chato,Azure方案Demo，调用的时候请自行注意llm的token限制\n
需要调用其他的llm也是
